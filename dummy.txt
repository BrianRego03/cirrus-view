âœ… Option 2: Periodic token validity check

If you want to proactively check if the session is still valid every X minutes:

useEffect(() => {
  const interval = setInterval(() => {
    fetch("/userState", { credentials: "include" })
      .then(res => {
        if (!res.ok) throw new Error();
        return res.json();
      })
      .then(data => setUser(data))
      .catch(() => {
        setUser(null);
        navigate("/login");
      });
  }, 5 * 60 * 1000); // Every 5 minutes

  return () => clearInterval(interval);
}, []);

    ðŸ” This revalidates the token periodically
    âœ… Still only 1 API call every few minutes
    ðŸ” Useful for banking apps, admin panels, etc.
